# Beamline-project

Python simulation for our proposal for the 2026 CERN Beamline For Schools competition. Our team comes from EEB2 in Brussels, Belgium.

As a part of our proposal on muon tomography, or muography, this code calculates the "perfect" standard deviation of a muon passing through different materials using the highland formula. It then simulates a normal distribution with a random number generator to simulate the scattering angle of the muons. This aims to to simulate the particle accelerator beam at CERN at very basic level for our initial proposal.
